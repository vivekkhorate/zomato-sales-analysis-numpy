# Zomato Sales Analysis using NumPy

This project analyzes restaurant sales data using **NumPy** and visualizes insights using **Matplotlib**.

## Project Overview
The project analyzes multi-year sales data of restaurants and performs statistical operations like:

- Total sales per year
- Minimum and maximum sales
- Average sales analysis
- Cumulative sales growth
- Data visualization

## Technologies Used
- Python
- NumPy
- Matplotlib
- Jupyter Notebook

## Dataset Structure

Sales data format:

[Restaurant_ID, 2021, 2022, 2023, 2024]


## Key NumPy Operations Used

- np.sum()
- np.min()
- np.max()
- np.mean()
- np.cumsum()

## Visualization

Matplotlib is used to plot **average cumulative sales growth across restaurants.**

## Project Output

The project calculates:

- Yearly total sales
- Restaurant performance
- Cumulative sales trends
